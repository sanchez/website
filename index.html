Projects

Hermes Development

Hermes is a nodejs application for converting markdown to pdf, or latex. The main goal of the project is to allow users to easily extend an existing template and create their own. It is also hoped that in the future, this program will write directly to pdf, allowing users with more flexibility in the way that data is handled and presented.

Titan Planning

The lightweight text editor, with a beast under the hood. There isn't a lot of text editors out there that are lightweight and allow plugins and extensions (apart from vim). The idea behind Titan is to be as lightweight, and extendable as vim or neovim, but be built on nodejs and provide beautiful and easy to understand documentation and API.
